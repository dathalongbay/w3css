Thuộc tính css overflow ( tràn )
        The CSS overflow property controls what happens to content that is too big to fit into an area.
        Thuộc tính css overflow kiểm soát điều gì sẽ xảy ra khi nội dung quá lớn không vừa vào trong thẻ html 

        Các giá trị của css overflow :
        - visible ( hiển thị phần tràn ra ngoài ) giá trị mặc định 
        - hidden ( ẩn tràn ra ngoài)
        - scroll ( hiển thị thanh scroll chuột bất kể nội dung có tràn ra ngoài hay ko)
        - auto ( hiển thi thành scroll khi nội dung tràn ra ngoài và không hiển thị thanh scroll nếu nội dung không tràn ra ngoài)
