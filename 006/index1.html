CSS cho thẻ a 
        - a:link ( thông thường )
        - a:visited ( đã truy cập )
        - a:hover ( di chuột vào thẻ a)
        - a:active ( đang click chuột vào thẻ a)

        khi viết css cho thẻ a phải viết theo quy tắc 
        link => visited => hover => active
